DISEÑO EDITORIAL· DISEÑO GRÁFICO· DISEÑO WEB·

IRIS AMADOR

Esta es mi web de autora para los libros románticos que he autopublicado. Su función básica y principal es actuar como una landing para que lectoras puedan subscribirse fácilmente a la newsletter. A este fin, el formulario de subscripción es lo primero que se ve en abrir la web.

Es una web simple de dos páginas construida con HTML, CSS y Javascript. El "sobre mí" de la autora está construida dentro del mismo HTML que el formulario para reducir el número de páginas, aprovechar y evitar copiar-pegar tanto HTML y mejorar SEO, pero se esconde de vista mediante Javascript a menos que el usuario quiera verlo. La segunda página, un listado de los libros publicados, está aparte para que el número de imágenes no ralentice la carga de la página principal si el usuario tiene una conexión lenta de internet.

También he diseñado el logo, ajustado al género y al uso, y he creado contenido para redes sociales.
