This is my author website for self-published romance books. Its basic and main function is to act as a landing page for readers to easily subscribe to the newsletter. To this end, the subscription form is the first thing one sees when opening the website.

It is a simple two-page website built with HTML, CSS and Javascript. The author's "about me" is built within the same HTML as the form to reduce the number of pages, take advantage of and avoid copy-pasting so much HTML and improve the SEO, but it is hidden from view using Javascript unless the user wants to see it. The second page, a list of published books, is a new HTML document so that the number of images does not slow down the loading of the main page if the user has a slow internet connection.

I have also designed the logo, adjusted to the genre and use, and I have created content for social media.
